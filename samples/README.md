# Synthetic Demonstration Report

This directory contains a strictly synthetic demonstration report designed to test the MediLens AI extraction and classification pipeline.

**IMPORTANT: This is entirely synthetic data. It is NOT a real medical report and must never be treated as medical data.**

## Generating the Report

The report can be regenerated by running the python script:

```powershell
python samples/generate_sample_report.py
```

## How to use

1. Open the Streamlit application.
2. Upload `sample_lab_report.pdf`.
3. Click "Extract Report Text".

## Expected Detected Statuses

The application deterministic rules engine should detect the following:
- **Hemoglobin**: `LOW` (11.2 < 12.0)
- **Glucose**: `NORMAL` (70 <= 96 <= 99)
- **Platelets**: `NORMAL` (150 <= 250 <= 450)
- **TSH**: `HIGH` (4.8 > 4.0)
- **Ferritin**: `NORMAL` (18 > 15)
